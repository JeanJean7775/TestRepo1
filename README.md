Credit Risk is an example of use of Random Forest Classifier 
The data were uploaded from Kaggle, with 4 variables Loan, Age , Income and Default
A troughout analysis showed few missing data and not an import need to clean data (unlike in real world)
Exploraty Analysis of the relationship between default and bins loans and age showed early path signs
Individuals older than 35 did not default their loans and the higher the loan , the higher the possibility of default
The accuracy score was of 98%, with the confusion matrix at 0.99
